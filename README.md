# wikicord
Wikipedia bot for Discord

[**Invite me!**](https://discord.com/api/oauth2/authorize?client_id=1093013007329665105&permissions=292057869312&scope=bot)

## Commands
- ``$s + <search query>:`` Search a content
- ``$q:`` Help
- ``$a:`` About this bot

## Commands
- `-s <search query>`: Search a content
- `-ss <language> <search query>`: Search a content in another language
- `-q`: Help
- `-l`: Link to the full list of all supported language codes of Wikipedia
- `-d`: Donate me
- `-a`: About this bot
